How Well Can Machine Learning Models Predict the Price of Bitcoin?

Forecasting Bitcoin Prices via ARIMA, XGBoost, Prophet, and LSTM models in Python

Article Comparing the Machine Learning Models

In this article, I will test the Bitcoin forecasting abilities of 4 different Machine Learning models in Python: ARIMA, Prophet, XGBoost, and LSTM. By splitting the data into a testing and training set, I will compare each model’s performance with one another and conclude which performed best. I will be using the Root Mean Squared Error (the standard deviation of the residuals) of each model’s test to determine the best model.

Time Series Forecasting in Python

Each model was trained on the first 70% of the time series and then tested on the remaining 30% (439 days) against its predicted values. Firstly, I applied the Prophet Model developed by Facebook, then an ARIMA Model (AutoRegressive Integrated Moving Average), then a neural network Long Short-Term Memory model, and finally a XGBoost Model. The deep learning network, the LSTM model, proved to have the lowest overall RMSE and was therefore the best.
