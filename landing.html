Spotify Typing Test App

Full stack web application that transforms the lyrics of any Spotify song into a typing test as the song plays

An Engaging and Fun Way to Practice Typing

I used to jam out to music while taking online typying tests to work on my typing speed/accuracy. However, I found myself getting bored and switching to just listening to music instead. This is when I came up with the idea to combine both the lyrics from my music and the typing test into one! I created this full stack web application in JavaScript that transforms the lyrics of any Spotify song into a typing test as the song plays!

Technologies Behind the App

I used Spotify’s Web API, React, Node.js, and Express to develop this app. Unfortunately Spotify doesn't allow their API to be used publicly for Web Apps considered "games". Therefore the site can only be accessed if you send me the email address you've used for your spotify account. Feel free to email me at aaronfloreani@gmail.com if you'd like access to the site!
